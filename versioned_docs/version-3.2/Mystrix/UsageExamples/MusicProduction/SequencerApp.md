# Sequencer App

The <DocLink to="/docs/MatrixOS/Applications/Sequencer">Sequencer App</DocLink> is a powerful standalone sequencer built into Matrix OS.

It pairs especially well with the <DocLink to="/docs/MatrixOS/Applications/Note">Note App</DocLink> for composition, arranging, and live performance.
