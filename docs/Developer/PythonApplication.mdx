# Code Your Application (Python)

## Ways to Use Python

### Matrix OS Developer Toolkit

Use [Matrix OS Developer Toolkit](./MatrixOSDeveloperToolkit) for the fastest edit-test loop. The Python panel can run short snippets and staged `.py` files before you copy an app to device storage.

### Python REPL

Open the Python app from the application launcher and connect over USB serial. The REPL is best for quick experiments, API exploration, and testing short snippets.

### Standalone Python App

Create a folder app on device storage:

```text
/MatrixOS/Applications/<appname>/
```

The folder must contain:

- `AppInfo.json`
- one or more `.py` files
- the main file referenced by `appMainFile`

:::warning Storage Required
Standalone Python applications require device storage. On Mystrix devices, install a microSD card first. See [Install Micro SD Card](/docs/Mystrix/InstallMicroSD).
:::

## AppInfo.json

```json
{
  "name": "Hello World",
  "author": "203 Systems",
  "color": [255, 0, 0],
  "version": 1,
  "osMinimalVer": [4, 0, 0],
  "appMainFile": "main.py"
}
```

Required fields:

- `name`: display name in Matrix OS
- `author`: author or organization
- `color`: app icon color as `[R, G, B]`
- `version`: app version number
- `osMinimalVer`: minimum Matrix OS version
- `appMainFile`: Python file to run

## API Style

Python APIs use `snake_case` names and Python values:

- Colors are packed RGB integers or RGB/RGBW tuples.
- Points are `(x, y)` tuples.
- Input IDs are `(cluster_id, member_id)` tuples.
- Input events and snapshots are dictionaries.
- `MatrixOS.Input` provides keypad, touchbar, and other input events.

## Minimal Example

```python
import MatrixOS

MatrixOS.LED.clear()
MatrixOS.LED.set_xy(0, 0, (255, 0, 0))
MatrixOS.LED.update()
```

Expected result: LED `(0, 0)` turns red.

## Input Example

```python
import MatrixOS

for _ in range(100):
    event = MatrixOS.Input.get_event(10)
    if event:
        point = event.get("point")
        keypad = event.get("keypad")
        if point and keypad and keypad["pressed"]:
            MatrixOS.LED.set_xy(point[0], point[1], (255, 255, 255))
            MatrixOS.LED.update()
    MatrixOS.SYS.task_yield()
```

Expected result: during the short polling window, pressing a key lights the matching LED white.

## Try It In Matrix OS Developer Toolkit

1. Start Matrix OS Developer Toolkit from the Matrix OS repository.
2. Open the WebUI and switch to the Python panel.
3. Paste the minimal example or upload a `.py` file.
4. Run the script and check the LED panel or Python output.

For automation, the toolkit JSON-RPC bridge can run snippets with `python.runText`; see [Matrix OS Developer Toolkit](./MatrixOSDeveloperToolkit#json-rpc-bridge).

## Deploy To A Device

1. Create `/MatrixOS/Applications/<appname>/` on device storage.
2. Copy `AppInfo.json` and `main.py` into that folder.
3. Restart Matrix OS or reopen the application launcher so the app list refreshes.
4. Launch the app from the application launcher.

## Debug

Use `print()` for app output and keep test loops bounded while debugging. See [Debug Your Application (Python)](./DebugMatrixOSPython).
